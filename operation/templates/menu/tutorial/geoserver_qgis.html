Tutorial menggunakan GeoServer di QGIS


  1. Buka QGIS Desktop

    2. Buka QGIS di desktop, disarankan untuk menggunakan QGIS versi 3.16 ke atas untuk menghasilkan koneksi data yang baik. Jika QGIS belum terinstall di komputer, aplikasi ini dapat diunduh secara gratis di sini



  2. Buka Open Data Source Manager

    3. Klik icon Open Data Source Manager yang biasanya ada di toolbar sisi kiri atas



  3. Pilih menu WFS

    4. Pilih WFS / OGC API - Features pada sidebar



  4. Membuat koneksi WFS baru

    5. Klik tombol New pada menu WFS / OGC API - Features



  5. Setup koneksi ke Kaleka Satu Data

    6. Tuliskan Nama koneksi dengan nama Kaleka Satu Data. Kemudian isikan URL dengan kode berikut ini. Jangan lupa untuk memilih versi WFS 1.1 pada WFS Option di bawah.



  6. Otentikasi koneksi ke GeoServer

    7. Isikan Username dan Password pada form Basic Authentication. Kontak ke afaiz@kaleka.id untuk mendapatkan username dan password GeoServer.



  7. Tambahkan layer WFS ke dalam halaman peta

    8. Klik Connect untuk mengkoneksikan dengan GeoServer Kaleka Satu Data. Pilih layer yang akan ditampilkan di QGIS kemudian klik Add. Jika membutuhkan dta yang spesifik, dapat dilakukan filter data dengan tombol Build query



  8. Selamat, kamu sudah bisa menggunakan data dari Kaleka Satu Data di QGIS
